1°Abrir o VSCODE
2°No terminal digitar: cd desktop
3°No terminal dentro do desktop: mkdir "projeto"
4° No terminal: cd projeto
5° dentro da pasta projeto: code .
6° no terminal da pasta projeto: touch index.html
7° no terminal: mkdir src
8° no terminal: cd src
9° no terminal: mkdir css
10° no terminal: mkdir js
11° no terminal: cd css
12° no terminal:touch estilo.css
13° no terminal: cd ..
14° no terminal: cd js
15° no terminal: touch script.js
16° no terminal: cd ..
17° no terminal: cd .. até chegar na raiz do projeto
18° no terminal: git init
19° se estiver na fiap: git config --global user.name "seu primeiro nome"
20° se estiver na fiap: git config --global user.email fgarciamello@gmail.com
21° no terminal: git status
22° se estiver vermelho: git add .
23° se estiver verde: got commit -m "Mensagem do que voce fez"
24° entrar no github em repositorio, cria um novo repositorio, colocar um nome e deixar publico, clicar no botao criar repositorio(dica de ouro- nunca traduza a pagina do github)
25° no bloco que estiver as 3 linhas
    1°linha- conecta seu gitbash com o seu github
    2°linha- muda o nome master para main
    3°linha- pega tudo que estiver no projeto e sobe para o github
26° Caso voce esteja em outro pc: no seu repositorio no botao verde copie o link
27° abra o seu VSCODE no terminal cd desktop : git clone (copie o link), isso vai clonar seu projeto e pode continuar da onde parou
28° se modificar seu codigo faça todo o processo do
    git status
    git add.
    git commit -m "mensagem"
    e para subir as alteraçoes só digitar : git push
